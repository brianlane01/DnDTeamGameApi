About The Api

The DnD Creator Application's API is tailored to facilitate the creation and management of Dungeons and Dragons games seamlessly. It offers endpoints for both game creation and storage. Users can initiate new maps, characters, vehicles, and much more. The API accommodates saving and loading characters, and story development. It allows Dungeon Masters to create custom maps, encounter descriptions, and character details, enhancing gameplay immersion. Additionally, the API incorporates authentication and authorization layers to safeguard sensitive campaign data, ensuring a secure and collaborative platform for DnD enthusiasts to embark on epic adventures.
